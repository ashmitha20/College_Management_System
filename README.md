College Management System Web App

Project Overview

This College Management System is a web application built using Flask and SQLite. It allows users to manage college data, including students, faculty, and management members. The app provides a simple and user-friendly interface to perform CRUD operations (Create, Read, Delete) while ensuring secure user authentication.

Features

User Authentication for secure access
Add, view, and delete students, faculty, and management members
Clean, minimalistic user interface

Technologies Used

Backend: Python (Flask)
Frontend: HTML, CSS
Database: SQLite
